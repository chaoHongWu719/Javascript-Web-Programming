<header>是頁首的標籤，通常用來放置網站的標題、副標題等內容。

<nav>是導航的標籤，通常用來放置網站的主要導航連結。

<a>是超連結的標籤，href屬性用於指定網址，target可選_blank(在新分頁中開啟連結)、_self(在當前分頁中開啟連結)。

<p>是段落文字的標籤，剛剛的標題標籤有h1到h6可供使用，可用CSS來設定標題字體大小、顏色等格式。

<q>是引用文字的標籤，會用雙引號標註起來，不會縮排。
<blockquote>是引用段落的標籤，整段會有縮排，可以寫一大段文字。底下示範一些特殊符號的寫法。

<span>與<div>標籤可以用來對文字的一部分進行格式化，通常是為了設定文字樣式，差異在於<div>是區塊 (block) 元素，用於包一整塊段落；而<span>是行內 (inline) 元素，用來包一小段文字。

接下來來說明圖片的插入方式。<img> 標籤用於插入圖片，但這是 inline element/p>，通常會搭配 block element <figure>使用，這樣就可以在圖片下方加上說明文字<figurecaption>

<audio>標籤用於插入音訊，通常會搭配<source>標籤使用，可以提供多種格式的音訊檔案，以確保在不同瀏覽器上的兼容性。<audio>標籤也可以使用controls屬性來顯示播放控制器，讓使用者可以播放、暫停、調整音量等操作。

<video>標籤用於插入影片，用法同<audio>標籤。

接著介紹如何用html繪製表格，需要使用<table>、<tr>、<td>等標籤，其中<tr>標籤用於建立表格的列 (row)，<td>標籤用於建立表格的欄 (column)。 <caption>則可以用來為表格添加標題，<summary>則顯示給瀏覽器看的資訊 (使網頁更容易被搜尋到)。 <thead>、<tbody>、<tfoot>則分別用於定義表格的表頭、表身和表尾，這樣可以做到 incremental rendering (漸進式產生body內文，用於資料量大，此時<tfoot>需要寫在<tbody>前方)，且方便滾動 (固定顯示 head 和 foot)。 更進階的標籤如<rowspan>、<colspan>則是用來合併儲存格。

最後介紹表單 (form)。請注意:一個網頁可以不只一個 form, 但一次只會有一個 form 生效

  • <form> 標籤用於建立表單，可以包含多個屬性
    • <name>, <id> 用於指定表單的名稱
    • <action> 用於指定表單提交的 URL
    • <method> 用於指定表單提交的 HTTP 方法，html 可用
      • GET: 把表單所有資訊都放在 url 中，再傳送給伺服器 e.g. submit.php?account=user1&password=user123...
      • POST: 先傳送特定表頭給伺服器，再傳送真正的表單資料。適合有密碼 (比較安全) 或大表單 (網址有長度限制)
    • <enctype> 用於指定表單資料的編碼方式，尤其在上傳多個檔案時需要設定
  • <input> 用於建立表單元素
    • <label> 用於為表單元素提供文字標籤，可用 for = "elementID" 屬性來指定元素；也可以在元素中添加屬性 label = "labelID"
    • <type> 用於指定元素的類型 (text, password, select, checkbox, radio, file, reset, submit...)
    • <value> 用於指定元素的初始值
  • 一些其他的屬性
    • placeholder: 用於指定元素的提示文字 (當元素為空時顯示)
    • checked: 用於指定 radio 或 checkbox 元素的預設選項
    • required: 用於指定元素為必填項
