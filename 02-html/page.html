<dl>
定義清單的開始 (沒有項目符號)
<dt>
定義清單項目的名稱 (沒有項目符號)
<dd>
定義清單項目的描述 (沒有項目符號)

<header>是頁首的標籤，通常用來放置網站的標題、副標題等內容。

<nav>是導航的標籤，通常用來放置網站的主要導航連結。

<a>是超連結的標籤，href屬性用於指定網址，target可選_blank(在新分頁中開啟連結)、_self(在當前分頁中開啟連結)。

anchor:交大課程時間表網址 跳轉到底部

<p>是段落文字的標籤，剛剛的標題標籤有h1到h6可供使用，可用CSS來設定標題字體大小、顏色等格式。

<q>是引用文字的標籤，會用雙引號標註起來，不會縮排。
<blockquote>是引用段落的標籤，整段會有縮排，可以寫一大段文字。底下示範一些特殊符號的寫法。
請注意:html並不會理會在編輯器上的連續空格或換行，因此需要使用pre標籤將文字做預格式化處理，如跳行需使用<br>標籤，一行線條則需使用<hr>標籤。
  
                這是預格式化的文字，會保留空白和換行。以下是一個表格範例。
                html code       符號　　說明　　
                -----------------------------------------------------------
                &lt;              <          小於
                &gt;             >          大於
                &amp;          &          和號
                &copy;         ©          版權符號
                &quot;          "          雙引號

<span>與<div>標籤可以用來對文字的一部分進行格式化，通常是為了設定文字樣式，差異在於<div>是區塊 (block) 元素，用於包一整塊段落；而<span>是行內 (inline) 元素，用來包一小段文字。

接下來來說明圖片的插入方式。<img> 標籤用於插入圖片，但這是 inline element/p>，通常會搭配 block element <figure>使用，這樣就可以在圖片下方加上說明文字<figurecaption>

<audio>標籤用於插入音訊，通常會搭配<source>標籤使用，可以提供多種格式的音訊檔案，以確保在不同瀏覽器上的兼容性。<audio>標籤也可以使用controls屬性來顯示播放控制器，讓使用者可以播放、暫停、調整音量等操作。

<video>標籤用於插入影片，用法同<audio>標籤。

<footer>是頁腳的標籤，通常用來放置版權聲明、聯絡資訊等內容。
