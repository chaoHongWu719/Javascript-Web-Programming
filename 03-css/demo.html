03- CSS practice

基本介紹

no show

CSS 是 Cascading Style Sheets 的縮寫，用於控制網頁的樣式。有以下三種寫法:

  1. inline: 直接寫在 html 標籤內 e.g. <p style = “font-size: 24pt;”>
  2. internal: 寫在 html 檔案中 <head> 內，使用 <style> 標籤
  3. external: 寫在另一個獨立的檔案中，最為推薦 e.g. <link rel = "stylesheet" type = "text/css" href = "styles.css">

CSS 的語法為: selector { property1: value1; property2: value2; ...}， 而註解則是 /* comment */ selector 是用來選擇要套用的 html 標籤，主要可分為三類，越下層的 selector 會覆蓋上層的 selector:

  1. type selector: 選擇特定的 html 標籤 e.g. body
  2. group selector: 選擇多個 html 標籤 e.g. h1, h2, h3
  3. universal selector: 選擇所有 html 標籤 e.g. * (萬用字元)
  4. id selector: 選擇特定的 id e.g. #id
  5. class selector: 選擇特定的 class e.g. tagName.className or *.className
    注意: 一個 tag 可以有多個 class (e.g. <div class = "class1 class2">)，而 CSS 中的 class selector 可以寫為 tag.class1.class2 (同時符合 class1 和 class2) 或 tag.class1, tag.class2 (符合 class1 或 class2)
    比較: id 在一個 html 中只能出現一次 (較少寫在 external CSS 中，可以直接用 inline CSS), class則可以不同 tag 重複使用

字型設定

no show

font 有以下常見屬性，若有使用中文字體，在 CSS 開頭須加上 @charset "UTF-8"; 以確保中文能正常顯示。

  1. font-family: 依照指定順序設定字型，通常英文字型寫前面，中文寫後面 e.g. font-family: "Times New Roman", "標楷體";
    若想要使用 google 的 Noto 系列字型，可以在 <head> 中加入以下程式碼來匯入字型:<link href = 字型網址 rel = "stylesheet">
  2. font-size: 字體大小 e.g. font-size: 24pt;，其中單位有 px (pixel), em (大寫M大小), ex (小寫x大小, 約為 em 一半), %
    也可使用 font-size: xx-large;，有 xx-small, x-small, small, medium, large, x-large, xx-large
    若有巢狀tag, 則可將 child tag 設定為 larger/smaller
  3. font-weight: 字體粗細, 有 normal, bold, bolder, lighter, 100~900 e.g. font-weight: bold;
  4. font-style: 字體樣式 e.g. font-style: italic;
  5. line-height: 航高，單位與 font-size 類似 e.g. line-height: 1.2em;
  6. text-align: 文字水平對齊方式 e.g. text-align: center;
  7. text-indent: 首行縮排 (+:縮排; -: 凸排) e.g. text-indent: 2em;
    小技巧 (用於畫線但不顯示文字): <p style = "text-indent: -3600px; height: 0.2em; background-color: green;">no show</p>
  8. text-decoration: 文字裝飾 e.g. text-decoration: underline; (底線) 或 text-decoration: line-through; (刪除線)
  9. text-shadow: 文字陰影 (horizontal offset, vertical offset, blur radius, color) e.g. text-shadow: 2px 2px 4px gray;

顏色與背景

no show

顏色有以下幾種寫法:

  • RGB (16進位, 0-255) e.g. color: #ff0000; (紅色)
  • rgb(R, G, B) e.g. color: rgb(0, 255, 0); (綠色) 或 color: rgba(0%, 0%, 100%); (藍色)
  • rgba(R, G, B, 透明度) e.g. color: rgba(255, 0, 0, 0.5); (半透明紅色)
  • hsl(色相, 飽和度, 亮度) e.g. color: hsl(0, 100%, 50%); (紅色)
  • hsla(色相, 飽和度, 亮度, 透明度) e.g. color: hsla(0, 100%, 50%, 0.5); (半透明紅色)
    其中色相 (hue) 的範圍為 0-360 度, 0/360 為紅色, 120 為綠色, 240 為藍色

背景圖片添加: 使用 background-image: url("圖片路徑")，可搭配

  • background-repeat: repeat-x or repeat-y or no-repeat;
  • background-position: right bottom ..
  • background-size: 200px 100px;
  • background-attachment: fixed; (背景固定，視窗滑動仍在固定位置)
  • background-image 可用漸層色 e.g. background-image: linear-gradient(to right, red, blue);
